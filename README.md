Matrixportal weather code. you will need to add a "secrets.py" in this form:

```
secrets = {
        'ssid' : 'your wifi SSID',
        'password' : 'wifi password',
        'openweather_token' : 'token for openweather api',
        'aio_username': 'adafruit username',
        'aio_key': 'adafruit key',
        'city': 'Chicago, US'
}
```
